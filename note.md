# Полезные поля для AI помощника

## Структура заметки (JSON)

```json
{
  "id": "uuid-1234",                           // Уникальный ID
  "type": "task",                              // task, reminder, idea, note
  "text": "Купить хлеб",                       // Содержание
  "created_at": "2026-06-25 11:11:25",        // Когда создана
  "updated_at": "2026-06-25 11:11:25",        // Последнее изменение
  
  // Для приоритизации
  "priority": "high",                          // high, medium, low
  "deadline": "2026-06-26",                    // Срок выполнения
  "status": "pending",                         // pending, in_progress, done
  
  // Для организации
  "category": "shopping",                      // Категория/тег
  "tags": ["urgent", "personal"],              // Множественные теги
  
  // Для AI анализа
  "context": "Нужен для завтрака",            // Контекст/описание
  "energy_level": "low",                       // low, medium, high (для ADHD)
  "time_estimate": 30,                         // Минут на выполнение
  "difficulty": "easy",                        // easy, medium, hard
  
  // Для напоминаний
  "remind_at": "2026-06-25 09:00:00",         // Когда напомнить
  "recurring": "daily",                        // daily, weekly, monthly, none
  
  // Для AI порекомендований
  "completed_at": "2026-06-25 12:30:00",      // Когда завершено
  "is_archived": false,                        // Архивирована?
  "notes": "Помидоры берёшь спелые"            // Дополнительные заметки
}
```

## Самые важные поля для AI

| Поле | Назначение |
|------|-----------|
| **priority** | AI рекомендует срочные задачи |
| **energy_level** | Для ADHD - подбирает задачи под текущее состояние |
| **time_estimate** | AI подбирает задачи по времени на выполнение |
| **deadline** | Помогает с планированием и организацией |
| **tags** | Организация, поиск и фильтрация |
| **completed_at** | Анализ продуктивности и паттернов |
| **status** | Отслеживание прогресса |
| **difficulty** | Балансировка сложности задач |

## Использование в приложении

Эти поля позволяют AI помощнику:
- 📊 Анализировать продуктивность
- ⚡ Подбирать задачи под текущую энергию
- ⏰ Планировать день оптимально
- 🎯 Приоритизировать важное
- 🔄 Выявлять паттерны поведения
- 💡 Давать персонализированные рекомендации
